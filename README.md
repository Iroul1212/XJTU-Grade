# 篡改猴安装及离线脚本导入教程

### 第一阶段：环境准备（安装篡改猴）

在使用脚本前，必须先在浏览器中安装运行环境。请根据您使用的浏览器选择对应步骤。

#### 1. 打开扩展商店

* **Microsoft Edge 浏览器（推荐）：**
* 点击浏览器右上角的三个点 `...` 图标。
* 选择 **“扩展”** -> **“管理扩展”**。
* 点击 **“获取 Microsoft Edge 扩展”**。


* **Google Chrome 浏览器：**
* 直接访问 Chrome 应用商店（需网络环境支持）。



#### 2. 搜索并安装

* 在商店搜索框输入：`Tampermonkey`或`篡改猴`
* 找到目标应用：图标为**黑色背景、有两个圆圈眼**的猴子形象。
* 点击 **“获取”** 或 **“添加至浏览器”** 按钮。
* 等待浏览器弹出提示，确认安装完成后，浏览器右上角会出现篡改猴的小图标。

### 第二阶段：导入离线脚本

#### 1. 复制脚本代码

* 如果您收到的是 **`.js` 文件**：请使用电脑自带的“记事本”或任意文本编辑器打开该文件。
* 如果您收到的是 **一段文本**：请直接选中。
* **操作：** 全选内容（快捷键 `Ctrl + A`），并复制内容（快捷键 `Ctrl + C`）。

#### 2. 创建新脚本

* 点击浏览器右上角的 **篡改猴图标**。
* 在弹出菜单中，选择 **“添加新脚本”**（图标通常为一个带有加号 `+` 的文件纸）。

#### 3. 植入代码（关键步骤）

* 此时会打开一个编辑器界面，里面会有默认生成的十几行代码。
* **重要：** 请先**全选并删除**编辑器内原有的所有内容，确保编辑区是**空白**的。
* **粘贴：** 将第 1 步复制的代码粘贴到空白编辑区中（快捷键 `Ctrl + V`）。

#### 4. 保存生效

* 点击编辑器左上角的 **“文件”** 菜单，选择 **“保存”**。
* 或者直接使用键盘快捷键 `Ctrl + S`。
* 保存后，编辑器页面通常会自动关闭，表示脚本已成功载入系统。

### 第三阶段：验证与使用

1. **访问目标网站：** 打开该脚本所服务的网页。
2. **检查状态：** 观察浏览器右上角的篡改猴图标。
* 如果图标上出现**红色的数字角标**（例如 `1`），说明脚本已成功识别并在当前页面运行。

3. **功能测试：** 根据脚本的功能描述，检查网页是否发生了预期的变化。
